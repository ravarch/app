# React SPA

**React SPA** powered by **Cloudflare Workers**. It leverages Cloudflare's Asset handling to serve a blazing-fast frontend while exposing a serverless API from the same Worker instance.

> **Status:** Active Development
> **Stack:** React 19, Vite, Tailwind CSS 4, Cloudflare Workers, Agents SDK, MCP
